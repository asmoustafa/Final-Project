Image Resize

May 2022

Built an image resizing program using a seam-carving algorithim. Focused on testing, debugging, pointers, arrays, strings, streams, IO, and abstract data types in C++. The program cropped images in a content-aware fashion, creating new images of altered sizes while maintaining the same original components. The program sifts through the image to find the "seams", pathways of the least important pixels, and removes them.
